# Color Palette Maker

Web app created for designers, helps with matching colors and generating color palettes in an easy way.

Color Palette Maker was designed with Responsive and Mobile First approach.

## Getting started
``` bash
$ npm install
```
``` bash
$ npm run start:dev
```

## Technologies used

* JavaScript
* Sass
* HTML
* Webpack
